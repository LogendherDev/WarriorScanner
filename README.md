Install the dependencies.

```sh
pip install -r requirements.txt
```

## How to use


This Program will work on cross platforms,

Note : Make sure you have bash command in environmental variable..
```sh
Windows
py runme.py

Linux
python3 runme.py (or) python runme.py
```
